# API-INTEGRATION-AND-DATA-VISUALIZATION

## Objective

Fetch weather data from OpenWeatherMap API and visualize temperature trends using Matplotlib.

## Technologies

- Python
- OpenWeatherMap API
- Matplotlib

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python api_visualization.py
```

## Output

- Weather data fetched from API
- Temperature trend chart generated
- dashboard.png saved
